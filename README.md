# Playwright + pytest UI Automation Framework

A comprehensive end-to-end testing framework using Playwright, pytest, and Python.

## Features

- **Page Object Model** for clean, maintainable test code
- **Parallel test execution** support with pytest-xdist
- **Comprehensive logging** and reporting
- **Environment configuration** management
- **Screenshot on failure** for debugging
- **Parameterized tests** and test markers
- **Type hints** for better IDE support

## Prerequisites

- Python 3.8+
- Node.js (for Playwright installation)
